# Your fayz.ai app

This project was created with [fayz.ai](https://fayz.ai).

## Editing

- **In fayz.ai** — open your project and describe the changes you want. Edits are committed to this repo automatically.
- **Locally** — clone the repo, then:

```sh
npm install
npm run dev
```

Pushed changes are reflected back in fayz.ai.

## Tech stack

Vite · TypeScript · React · Tailwind CSS · shadcn/ui

## Deploying

Open your project in [fayz.ai](https://fayz.ai) and click **Publish**. To use a custom domain, go to **Project → Settings → Domains**.
